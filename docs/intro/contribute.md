# 如何参与

欢迎为本项目贡献内容！

## 贡献方式

1. **Fork 本仓库**：点击 GitHub 右上角 Fork 按钮
2. **克隆到本地**：`git clone https://github.com/mengjio007/my-wiki.git`
3. **创建分支**：`git checkout -b feature/your-topic`
4. **编辑文档**：在 `docs/` 目录下修改或新增 Markdown 文件
5. **本地预览**：运行 `mkdocs serve` 查看效果
6. **提交 PR**：推送分支并创建 Pull Request

## 文档规范

- 使用 Markdown 格式编写
- 代码块请注明语言类型
- 保持目录结构清晰
